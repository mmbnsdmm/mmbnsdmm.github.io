# vuecli3

## cnpm
```
npm install -g cnpm --registry=https://registry.npm.taobao.org
```

## vue ui
```
cnpm install -g @vue/cli
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
